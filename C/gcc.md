gcc/g++

## gcc

```
cc = gcc      // gcc支持多种编程语言的编译器；

gcc -c main.c  -o main.o          //main.o编译生成的目标文件；

或 gcc -c main.c     ///生成目标文件 main.o;

```

## g++

```
g++ a.cpp   ///生成可执行文件a.out;

g++ a.cpp -o test   ///生成可执行文件test;

g++ -c a.cpp    ////生成目标文件a.o;

g++ a.o    ///生成可执行文件a.out;

g++ -c a.cpp -o t  ////生成目标文件t;

```
